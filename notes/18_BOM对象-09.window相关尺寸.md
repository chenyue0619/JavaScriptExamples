# window相关尺寸

* 分类

  * innerHeight 浏览器窗口的内容区域的高度，包含水平滚动条（如果有的话）=> 显示内容的可视区
  * innerWidth  浏览器窗口的内容区域的宽度，包含垂直滚动条(如果有的话)     => 显示内容的可视区
  * outerHeight 浏览器窗口的外部高度
  * outerWidth   浏览器窗口的外部宽度

* 举例

  ```html
  <script>
      console.log(window.innerWidth);
      console.log(window.outerWidth);
  </script>
  ```
