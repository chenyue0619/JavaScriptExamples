# 深克隆(递归函数)

* Array.isArray(arr);//判断是否是数组

* 克隆前后的数组不会藕断丝连

* 递归克隆数组

  ```html
  <script>
      let arrClone = [];
      let deepClone = function(arr){
          for(let i = 0;i < arr.length;i++){
              if(Array.isArray(arr[i])){
                  deepClone(arr[i]);
              }else{
                  arrClone.push(arr[i]);
              }
              // console.log(arrClone);
          }
          return arrClone;
      }
      console.log(deepClone([1,2,33,[45,656,[546,546,56,65,[566,56,6,5]]]]));
      </script>
  ```
