# HTTP状态码

* 状态码分类
  * 1**  服务器收到数据
  * 2**  请求成功，如200
  * 3**  重定向，如302
  * 4**  客户端错误，如404
  * 5**  服务端错误，如500
* 常见状态码
  * 200  成功
  * 301  永久重定向(配合location,浏览器自动处理)  Moved  Permanently
  * 302  临时重定向(配合location,浏览器自动处理)  Move Temporarily
  * 304 资源未被修改  Not Modified
  * 404 资源未找到
  * 403 没有权限
  * 500 服务器错误
  * 504 网关超时
