# 初识JSON

* JSON定义：

  * JSON全称JavaScript Object Nonation
  * Ajax发送和接收数据的一种格式

* JSON用处

  ```js
  //js -> JSON -> PHP -> Java
  //PHP/Java -> JSON -> JS
  ```

* JSON格式
  * 简单值形式
    * json文件后缀名.json
    * JSON的简单值形式就对应着JS中的基础数据类型
    * 数字、字符串、布尔值、null
    * undefined会转换成字符串类型
  * 对象形式
  * 数组形式
