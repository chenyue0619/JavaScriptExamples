### 20.isNaN（）函数检测下我们学过的数据类型是不是数值类型吧！

需要判断的数据如下：

> null，10，"我是小云同学"，"10"，undefined

一定要注意：isNaN()返回的是false，表示就是数值，返回的是true，表示就是非数值·

null => false

10 => false

"我是小云同学" => true

"10" => false

undefined => true
